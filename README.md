# 👗 نظام تأجير الفساتين

برنامج سطح مكتب متكامل لإدارة محلات تأجير الفساتين، مبني بـ Python + PyQt6 + SQLite

---

## 🚀 متطلبات التشغيل

- Python 3.9 أو أعلى
- نظام Windows / macOS / Linux

---

## 📦 خطوات التثبيت

### الخطوة 1 — تثبيت Python
1. اذهب إلى: https://www.python.org/downloads/
2. حمّل Python 3.11 أو أعلى
3. ✅ تأكد من تفعيل "Add Python to PATH" أثناء التثبيت

### الخطوة 2 — تحميل الملفات
انسخ مجلد `dress_rental` إلى جهازك.

### الخطوة 3 — تثبيت المكتبات
افتح Command Prompt (cmd) في مجلد المشروع وشغّل:

```
pip install -r requirements.txt
```

### الخطوة 4 — تشغيل البرنامج

```
python main.py
```

---

## 🗂 هيكل المشروع

```
dress_rental/
├── main.py              ← نقطة البداية

├── styles.py            ← تصميم واجهة المستخدم
├── requirements.txt     ← المكتبات المطلوبة
├── dress_rental.db      ← قاعدة البيانات (تُنشأ تلقائياً)
└── ui/
    ├── main_window.py   ← النافذة الرئيسية
    ├── dashboard.py     ← لوحة التحكم
    ├── dresses.py       ← إدارة الفساتين
    ├── customers.py     ← إدارة العملاء
    ├── rentals.py       ← إدارة التأجيرات
    ├── invoice.py       ← الفواتير
    └── reports.py       ← التقارير والإحصائيات
```

---

## ✨ مميزات البرنامج

### 🏠 لوحة التحكم
- إحصائيات فورية (فساتين متاحة، مؤجرة، متأخرة)
- إيرادات اليوم والشهر والمبالغ المعلقة
- تنبيهات التأجيرات المتأخرة
- تنبيهات مواعيد الإرجاع خلال 3 أيام

### 👗 إدارة الفساتين
- إضافة / تعديل / حذف الفساتين
- تصنيف بالمقاس، اللون، الفئة
- تتبع حالة كل فستان (متاح، مؤجر، صيانة)
- نظام كودات للفساتين

### 👥 إدارة العملاء
- تسجيل بيانات العملاء الكاملة
- البحث السريع
- سجل تأجيرات كل عميل

### 📦 إدارة التأجيرات
- تأجير فستان جديد بخطوات بسيطة
- نظام الخصومات والتأمين
- تتبع المدفوعات والمتبقي
- إرجاع الفساتين مع استلام الدفعات
- إلغاء التأجيرات

### 🧾 الفواتير
- فاتورة احترافية بالعربية
- طباعة مباشرة

### 📊 التقارير
- إيرادات شهرية
- أكثر الفساتين طلباً
- تقرير التأجيرات الشهرية
- إدارة المصروفات

## 🖥 إنشاء ملف exe (اختياري)

```

```
